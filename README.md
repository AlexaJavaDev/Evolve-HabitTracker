# 🌱 Evolve — трекер привычек

> Маленькие шаги каждый день приводят к большим изменениям.

**Evolve** — это простое и удобное веб-приложение, которое помогает формировать полезные привычки.  
Оно создано для тех, кто хочет стать лучше, но не знает, с чего начать. Добавляй привычки, отмечай прогресс и следи за своим ростом.

Проект написан на **Java Spring Boot** с использованием **Thymeleaf** и **Bootstrap**, и развёрнут локально через **H2 Database**.

## В разработке (ближайшие шаги)

- **➕ Добавление привычек**  
  Через простую форму можно создать привычку: указать название, описание, цвет и периодичность (ежедневно, по будням или по выходным). Данные сохраняются в базе H2.

- **📋 Список привычек**  
  Все привычки отображаются в виде аккуратных карточек на главной странице. Каждая карточка показывает название, описание, частоту и имеет цветовой акцент.

- **📊 Прогресс дня**  
  На главной странице отображается приветствие и прогресс-бар, который показывает, сколько привычек выполнено сегодня (пока заглушка, но скоро будет работать).

- **💾 Автосохранение**  
  Все изменения мгновенно сохраняются в базе данных — ничего не теряется при перезагрузке.

- **✅ Отметка выполнения**  
  Кнопка «Отметить» будет переключать статус привычки и обновлять прогресс-бар в реальном времени.

- **📈 Статистика**  
  Появится страница с графиками и календарём, где можно увидеть, как меняется твоя продуктивность по дням и неделям.

- **✏️ Редактирование и удаление**  
  Возможность изменить название, цвет или периодичность, а также удалить больше не нужную привычку.

- **🌙 Тёмная тема**  
  Для комфортной работы вечером.

## Технологии

Проект построен на современном стеке Java-разработки:

- **Java 17** — язык программирования
- **Spring Boot** — фреймворк для бэкенда
- **Thymeleaf** — шаблонизатор для генерации HTML
- **Bootstrap 5** — адаптивный и красивый интерфейс
- **H2 Database** — встроенная реляционная база данных (легко переключить на PostgreSQL)
- **Maven** — сборка и управление зависимостями
